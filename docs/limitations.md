# Limitations

This project should be understood as an autonomous research-assistance pipeline. It can discover, triage, summarize, organize, and propose candidate connections, but it does not replace mathematical proof, peer review, or expert judgment.

Important limitations:

- The agent is an assistant, not an independent scientist.
- Outputs require human review.
- Summaries may contain errors.
- Candidate hypotheses are speculative until reviewed.
- The public demo is a simplified representation.
- Sample artifacts are not production logs.
- The private repository contains the actual production pipeline and unpublished research materials.
- This repository intentionally excludes private prompts, private memory, raw traces, and internal scoring heuristics.

The right public claim is that the system supports a structured research workflow. It should not be described as independently proving new physics.
