# Public Roadmap

This roadmap is intentionally high level. It excludes unpublished research directions, private strategy, private prompts, and production implementation details.

Potential future additions:

- Add more sample artifact types.
- Add manually reviewed sanitized real daily runs.
- Add weekly synthesis artifact examples.
- Add a static dashboard for public run history.
- Add a paper provenance viewer for public artifacts.
- Add reproducible live public arXiv search, disabled by default.
- Add stricter validation rules.
- Add clearer distinction between accepted notes, rejected leads, and speculative candidate ideas.

Sanitized real runs should be added only after manual review.
