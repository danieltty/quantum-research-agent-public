# Claim Language

Use careful, accurate public language. The repository is meant to demonstrate workflow shape, artifact discipline, and a safe demo, not to expose the private production agent.

## Good Claims

- "I built a scheduled autonomous research assistant for quantum foundations."
- "The system produces provenance-backed daily research notes for human review."
- "The public repo contains sample artifacts, schemas, documentation, validation checks, and a reproducible demo mode."
- "The private production repo is not public because it contains unpublished research notes, prompts, scoring heuristics, operational configuration, and implementation details."
- "The system automates discovery, triage, summarization, and hypothesis-candidate generation, but final interpretation and research judgment remain human-led."

## Claims To Avoid

- "The AI independently discovers new physics."
- "The agent proves quantum theory."
- "The system is fully autonomous with no human judgment required."
- "This public repo is the full agent."
- "The sample artifacts are real production logs."
- "The demo scoring logic is the private production scoring logic."

## Suggested Short Description

Quantum Research Agent is a private autonomous research assistant for quantum foundations. This public repository documents the workflow shape and provides a safe demo of provenance-backed daily research artifacts without exposing private research IP.
